# Welcome to Remix!

## Deployment

Auto deployment via vercel.

## Development

To run locally,

```sh
yarn
```

Afterwards, start the Remix development server like so:

Install husky git hooks (IMPT!)

```sh
yarn prepare
```

```sh
yarn dev
```

Open up [http://localhost:3000](http://localhost:3000) and you should be ready to go!
